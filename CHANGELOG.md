# Changelog

## 0.2.0 — 2026-07-22

- Rewrote all normative guidance, examples, templates, and contribution material in English.
- Expanded the compact specification from 45 rules to cover read-only affordance, route state, stale data, progress, duplicate submission, performance, resilience, permissions, security, privacy, and localization.
- Added AI task-planning and delivery-report templates.
- Added detailed chapters for performance, permissions, localization, route state, and editability.
- Strengthened repository validation with English-only checks, rule-heading checks, version/changelog consistency, required chapter checks, and whitespace validation.

## 0.1.0 — 2026-07-22

- Established the AI agent entry point and rule precedence.
- Defined 45 framework-agnostic frontend rules.
- Covered layout, forms, overlays, navigation, feedback, responsive behavior, accessibility, and content.
- Added implementation workflow, validation checklist, deviation records, and project adoption templates.
- Added documentation-only repository validation to prevent UI implementation code from being introduced.
