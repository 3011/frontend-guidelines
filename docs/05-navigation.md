# 导航

## 可定位性

用户应能理解：

- 当前在哪里；
- 上一级是什么；
- 下一步可去哪里；
- 当前导航项为何处于激活状态。

当前状态不能只依赖颜色，应该同时通过形状、位置、文字或语义属性表达。

## Sidebar

展开状态下，文字已清晰可见时不重复显示 Tooltip。

折叠与展开必须作为完整状态转换设计：

1. 变更开始；
2. 宽度与内容动画；
3. 最终状态稳定；
4. 最终状态相关交互启用。

Tooltip、点击区域和文字显隐不能在错误阶段切换。动画过程中逐帧出现的闪烁、批量提示和点击错位都属于缺陷。

## 标签

导航标签应：

- 短；
- 使用用户熟悉的名词；
- 在不同页面保持稳定；
- 避免状态、数量和解释性长句混入名称。

## 移动端导航

移动端导航打开后不应遮挡无法恢复的上下文。选择目的地后应关闭，并将焦点移动到新页面的合理起点。
