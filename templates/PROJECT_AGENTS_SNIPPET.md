# Frontend Guidelines

进行任何前端开发前，必须阅读：

- `docs/frontend-guidelines/AGENTS.md`
- `docs/frontend-guidelines/FRONTEND_SPEC.md`
- 与当前任务相关的详细章节
- `docs/frontend-guidelines/docs/10-validation-checklist.md`

同时阅读项目根目录的 `PROJECT_FRONTEND_PROFILE.md`。

优先级：当前用户要求 > 本项目约束 > 项目已有合理模式 > 通用规范 > UI 库默认行为。

偏离通用规范时，必须在 `PROJECT_FRONTEND_PROFILE.md` 中记录规则 ID、原因、影响和验证方式。不得以组件库默认行为作为偏离理由。
